# France Top 50 Playlist Analysis Dashboard

This project analyzes the France Top 50 playlist to study:
- Audience sensitivity (explicit vs clean content)
- Content compliance
- Format preferences (single vs album)
- Song duration trends
  
Built using Python, Pandas, Plotly, and Streamlit.


## Features

- Interactive Streamlit Dashboard
- KPI Metrics:
  - Explicit Content Share
  - Clean Content Ratio
  - Single vs Album Ratio
  - Average Song Duration
  - Album Size Impact Index
  - Content Acceptance Score
- Trend Analysis over time
- Rank-tier comparison (Top 10 / 25 / 50)
- Business insights & recommendations

## Key Insights

- Clean content dominates higher rank tiers
- Singles outperform album tracks
- Songs around 3 minutes perform best
- Large albums may reduce track visibility

## Business Recommendations

- Prioritize clean content for France market
- Focus on single releases for better reach
- Maintain song duration between 2.5–3.5 minutes
- Avoid very large albums for playlist targeting
